### Git

Statut des fichiers dans le "stage"

```
git status

```

Ajout fichiers dans le "stage"

```
git add .

```

MAJ des fichiers dans le "stage"

```
git add -u

```

"Commit" du stage

```
git commit -m "commentaires sur les changements"

```

"Envoi" vers le repo "central"

```
git push  -u origin main

```
