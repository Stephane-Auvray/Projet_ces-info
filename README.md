## Dev

### Demarrage du Frontend

```
cd src
npm run dev

```

### Demarrage du Backend

```
npm start

```

### Test des APIs

`api.http`

### Git

Ajout fichiers dans le "stage"

```
git add .

```

"Commit" du stage

```
git commit -m "commentaires sur les changements"

```

"Envoi" vers le repo "central"

```
git push  -u origin main

```
